## Installation :zap:

 **1. Clone this repo by running the following command :-**
 ```bash
  git clone https://github.com/raihanislampolock/rms_tech_solutions.git
 ```
 
 **2. Now install all the required packages by running the following commands :-**
 ```bash
  npm install 
 ```
 **3. Now start the react server by running the following command :-**
 ```bash
  npm run build
  npm run watch
 ```
 **3. Create a `config.json` file in the project root folder from config.dev.json**
   - `config.json` file contains all the environment variables required for running the project.
   
 **4.** **🎉  Open your browser and go to  `http://your-domain:3005/`**
